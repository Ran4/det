# Det - Data gET

A command-line application to output and manage generated values.

Requires Python >= 3.6.

## Use

```bash
$ det get --random personnummer
19801019-2438
```

```bash
$ det stream personnummer
20101010-1234
19831409-3434
19731409-4210
...
^C
```

## Installing

Clone the repo. Run `./src/det.py`.
